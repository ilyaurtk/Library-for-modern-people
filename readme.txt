Это сайт для администрирования базы.
Соединение, редактирование и вывод из базы происходит с помощью библиотеки RedBeanPHP (\lib\rb.php)

Реквизиты:

*Установление соединения с базой local_user через логин mysql и пароль mysql 
(R::setup( 'mysql:host=localhost;dbname=local_users','mysql', 'mysql' );)

*В базе хранятся 2 таблицы (admin и users):

**admin: Содержит поля id login password. Эта таблица используется для хранения регистрационных данных администратора базы и вход на сайт происходит через логин и пароль администратора.

**users: Содержит поля id, uid, login, email, password, phone, points, taken_books, surname, name, otchestvo, num_pass, pass_serial.
Эти поля редактируются администратором базы. 


(Файл register.php по идее должен был быть предназначен для обычных пользователей, но на данный момент используется для ввода регистрационных данных пользователя.)
